# Export Columns

This document describes the main columns produced in CSV and Excel exports.

| Column | Description |
|--------|-------------|
| ASIN | Amazon product identifier |
| Brand | Parsed product brand |
| Title | Product title |
| Category | Amazon category |
| Price | Amazon price |
| Estimated eBay Price | Estimated selling price |
| Net Profit | Estimated profit after fees |
| Net Margin | Estimated margin |
| Upload Score | Overall upload priority |
| Safety Score | Risk evaluation |
| Profit Score | Profit evaluation |
| TitleQ | Title quality |
| ImageQ | Image quality |
| BulletQ | Bullet quality |
| DescQ | Description quality |
| SpecQ | Specification quality |
| Notes | Parser warnings or recommendations |
