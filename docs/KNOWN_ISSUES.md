# Known Issues

This document tracks known limitations in the current preview.

## Current limitations

- Amazon may occasionally present CAPTCHA or robot verification pages.
- Product page parsing depends on Amazon HTML and may require updates if layouts change.
- Estimated eBay profit is based on configured fees and should be verified before listing.
- Smart Queue is designed to prioritize candidates, not guarantee sales.

## Before v1.0 release

- Record any reproducible bugs here.
- Include steps to reproduce.
- Note expected vs actual behavior.
- Mark issues as resolved when fixed.
