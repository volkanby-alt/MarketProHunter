# MarketProHunter Post v1 Roadmap

This document lists ideas for after the first stable release.

## High priority
- eBay sold-price analysis
- Better duplicate detection
- Resume interrupted scans
- Automatic update checker

## Medium priority
- Saved scan profiles
- Custom export templates
- Advanced keyword scheduling
- Better image quality heuristics

## Long term
- Multi-marketplace support
- Historical pricing
- AI-assisted listing suggestions

The goal is to keep v1 stable and add these features only after real-world testing.
